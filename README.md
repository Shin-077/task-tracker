# Task Tracker - Angular Project

A simple and clean Task Management application built with **Angular** to understand the basic principles.

## Features
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Clean and responsive design
- Real-time updates

## Core Angular Concepts Demonstrated
- Standalone Components
- Two-way Data Binding (`ngModel`)
- Event Binding (`(click)`, `(change)`)
- Property Binding
- Directives (`*ngFor`, `*ngIf`)
- Component Logic in TypeScript

## How to Run

```bash
npm install
ng serve
